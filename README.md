SCENARIO 1 – K-NEAREST NEIGHBORS (KNN) 

Dataset link : https://www.kaggle.com/datasets/yasserh/breast-cancer-dataset
In this scenario, the K-Nearest Neighbors (KNN) algorithm is used for classification. The dataset is loaded, preprocessed, and scaled since KNN is distance-based. The data is then split into training and testing sets, and the KNN model is implemented to predict class labels. Finally, the model is evaluated using accuracy and confusion matrix to analyze its performance.


SCENARIO 2 – DECISION TREE CLASSIFIER

Dataset link : https://www.kaggle.com/datasets/ninzaami/loan-predication
In this scenario, a Decision Tree Classifier is used to predict whether a loan application will be approved or rejected using the Loan Prediction Dataset from Kaggle. The dataset is preprocessed by handling missing values and encoding categorical variables, then split into training and testing sets. The model is trained and evaluated using accuracy, precision, recall, and F1-score. Tree depth is adjusted to study overfitting, and visualizations like confusion matrix, tree structure, and feature importance are used to analyze performance.
